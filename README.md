
# **Boson Infinity — Windows Miner & Desktop Wallet**

### SHA-512 Proof-of-Work • Fixed 50,000,000 Supply • Testnet Release

This release contains two Windows applications:

* **boson-wallet-gui.exe** — desktop GUI wallet
* **miner_gpu.exe** — GPU mining client

Both programs are portable `.exe` files — **no installation required**.

---

# 📘 **1. Running the Desktop Wallet (boson-wallet-gui.exe)**

1. Double-click:

```
boson-wallet-gui.exe
```

2. The wallet will start a local web interface and open:

```
http://127.0.0.1:8090
```

3. On first launch, the wallet will:

* load your existing `wallet.json` **or**
* generate a new Ed25519 keypair
* create your Boson address (SHA-512(pub)[:20], 40-hex)
* save everything locally inside `wallet.json`

### ✔ Your private keys never leave your device.

Everything stays **offline** and is stored only in `wallet.json`.

# ⚡ **3. Mining BOS with RunMiner.bat**

### Users should **NOT** run the miner manually.

Instead, always use:

```
RunMiner.bat
```

This batch file launches the miner with the correct parameters.

---

## 🛠 **⚠️ IMPORTANT — Edit RunMiner.bat before starting the miner**

Open `RunMiner.bat` in Notepad and replace:

```
YOUR_WALLET_ADDRESS_HERE
```

with **your own wallet address** generated by the desktop wallet.

Example wallet address:

```
db92fe3a2720cc19256d9a1bab691346ef823e3a
```

That's the **only thing** the user must change.

---

## 📄 **RunMiner.bat — Template Included in Release**

The file looks like this:

```bat
@echo off
title Boson Infinity GPU Miner

REM -----------------------------------------------------
REM  EDIT THIS LINE ONLY → paste your own wallet address
REM -----------------------------------------------------
set WALLET=YOUR_WALLET_ADDRESS_HERE

REM Node URL (public testnet)
set NODE_URL=http://94.130.151.250

REM Public testnet mining API key
set MINER_KEY=ef5251f535ed7143b31398c1128d3b6a6ecbb011eb45f10205251f7192be1669

REM Oracle URL (for automatic energy reporting)
set BFI_ENERGY_ORACLE_URL=http://94.130.151.250:8090

REM Secret used to authenticate miner reports (internal)
set BFI_ORACLE_SECRET=e2c4dbe468705a18f20c83f94b5a174ccc5707ac314c19c2bebff1563c1d6b04

echo Starting Boson Infinity Miner...
miner_gpu.exe %NODE_URL% %MINER_KEY% %WALLET%
pause
```

### ✔ No power settings

### ✔ No energy price

### ✔ No fiat settings

### ✔ All energy metrics are handled automatically by the oracle

# ⛏️ **4. How the Miner Works**

Once you run `RunMiner.bat`, the miner:

1. Connects to the public node
2. Downloads mining work
3. Detects your GPU and power consumption automatically
4. Reports energy usage to the oracle
5. Submits valid blocks
6. Rewards are sent directly to your wallet address

### Expected console output:

```
[WORK] difficulty=34 reward=49.99 BOS
[FOUND] nonce=8233433
[SUCCESS] Block submitted!
```
